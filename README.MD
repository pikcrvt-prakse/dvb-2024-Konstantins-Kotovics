## Prakses pārskata struktūra un saturs

### 1. Prakses norises apraksts

Prakses vietas organizācijas apraksts:
Mana prakses vieta bija uzņēmumā Interum Technologies SIA, kas atrodas Rīgā, Artilērijas ielā 40i, 2. stāvā. Interum Technologies SIA ir specializēts IT uzņēmums, kas piedāvā plašu pakalpojumu klāstu, tostarp tīmekļa izstrādi, programmatūras izstrādi, datu analīzi un IT konsultācijas.

Darbības joma:
Uzņēmums nodarbojas ar dažādiem IT risinājumiem, kas aptver gan tīmekļa lietotņu izstrādi, gan klientu specifisko programmatūras risinājumu izveidi. Tas nodrošina augstas kvalitātes pakalpojumus vietējiem un starptautiskiem klientiem, piedāvājot pielāgotus risinājumus, kas atbilst klientu individuālajām vajadzībām. Es lielāko daļu laika pavadīju, izpētot un analizējot jau esošās vietnes, pievienojot jaunus elementus vai pārbaudot tos. 

Apstrādājamās informācijas plūsmu apraksts:
Informācijas plūsmas ietver klientu prasību analīzi, projektu pārvaldību, uzdevumu sadali starp komandas locekļiem un projektu izpildes uzraudzību. Informācijas apmaiņa notiek gan iekšēji starp darbiniekiem, gan ārēji ar klientiem, izmantojot dažādus komunikācijas rīkus un platformas. Visu informāciju par uzdevumiem es saņēmu no prakses vadītāja.

Paveiktā darba apraksts:
Prakses laikā izstrādāju vairākus projektus, izmantojot Laravel framework'u, kā arī veicu datubāzes pārvaldības uzdevumus, izmantojot SQL. Papildus tam apguvu darbu ar winSCP attālinātu serveru pārvaldībai un ieguvu praktisku pieredzi, strādājot ar dažādām programmēšanas tehnoloģijām.

### 2. Izmantojamās programmatūras apraksts

Laravel
- Raksturojums: PHP framework, kas paredzēts tīmekļa lietotņu izstrādei.
- Mērķis: Nodrošināt efektīvu un strukturētu tīmekļa lietotņu izstrādi, piedāvājot plašu funkcionalitātes klāstu.
- Izmaksas: Bezmaksas (atvērtā koda programmatūra).
- Īpašības: Nodrošina vieglu maršrutēšanu, migrācijas un datubāzes pārvaldību, augstu drošības līmeni.

winSCP
- Raksturojums: Grafiskais SFTP klients attālinātu serveru pārvaldībai.
- Mērķis: Droša datu pārraide starp lokālo datoru un attālinātu serveri, nodrošinot failu pārvaldību.
- Izmaksas: Bezmaksas.
- Īpašības: Atbalsta vairākus protokolus (SFTP, SCP, FTP), nodrošina drošu failu pārraidi.

SQL
- Raksturojums: Strukturētas vaicājumu valodas (SQL) datubāzes pārvaldības valoda.
- Mērķis: Nodrošināt efektīvu datu glabāšanu, pārvaldību un manipulāciju.
- Izmaksas: Atkarībā no izvēlētā SQL servera (MySQL, PostgreSQL ir bezmaksas). Izmantoju MySQL.
- Īpašības: Nodrošina kompleksu vaicājumu operācijas, strukturētu datu pārvaldību.

Adminer
- Raksturojums: Viegls un efektīvs datubāzes pārvaldības rīks.
- Mērķis: Vienkārša un ātra datubāzes pārvaldība, izmantojot tīmekļa interfeisu.
- Izmaksas: Bezmaksas.
- Īpašības: Viegls, uzstādāms vienā failā, atbalsta vairākas datubāzes.

Konsoles rīki
- Raksturojums: Komandrindas rīki dažādu uzdevumu veikšanai.
- Mērķis: Automatizēt uzdevumus un nodrošināt efektīvu izpildi.
- Izmaksas: Bezmaksas (atvērtā koda rīki).
- Īpašības: Nodrošina spēcīgas skriptu iespējas, plašu rīku klāstu.

Notepad++
- Raksturojums: Teksta un koda redaktors, kas atbalsta vairākas programmēšanas valodas.
- Mērķis: Koda rakstīšana un rediģēšana, nodrošinot sintakses izcelšanu un citas izstrādei nepieciešamās funkcijas.
- Izmaksas: Bezmaksas.
- Īpašības: Viegls, ātrs, atbalsta plašu spraudņu klāstu un sintakses izcelšanu vairākām valodām.

Composer
- Raksturojums: Atkarību pārvaldnieks PHP projektos.
- Mērķis: Instalēt un pārvaldīt bibliotēkas un pakotnes PHP projektos.
- Izmaksas: Bezmaksas.
- Īpašības: Automatizē atkarību instalāciju un atjaunināšanu, atvieglojot PHP projektu pārvaldību.

PHP Namespace Resolver
- Raksturojums: Spraudnis PHP koda rediģēšanai un namespace pārvaldībai.
- Mērķis: Automātiski pievienot un pārvaldīt namespace un use deklarācijas PHP failos.
- Izmaksas: Bezmaksas.
- Īpašības: Ātra namespace rezolūcija, automātiska use deklarāciju pievienošana.

Laravel Blade Snippets
- Raksturojums: Spraudnis koda fragmentiem Laravel Blade templating valodai.
- Mērķis: Nodrošināt bieži lietotu koda fragmentu ātru ievietošanu Laravel Blade failos.
- Izmaksas: Bezmaksas.
- Īpašības: Palīdz ātri ievietot un rediģēt Laravel Blade koda fragmentus, uzlabo produktivitāti koda rakstīšanā.

Laragon
- Raksturojums: Vietējais serveru pārvaldības rīks, kas nodrošina vieglu PHP, MySQL, Apache u.c. uzstādīšanu.
- Mērķis: Ātra un vienkārša vietējo serveru vides uzstādīšana un pārvaldība izstrādes vajadzībām.
- Izmaksas: Bezmaksas.
- Īpašības: Vienkārša uzstādīšana, intuitīvs interfeiss, atbalsta dažādas tehnoloģijas un rīkus.

phpMyAdmin
- Raksturojums: Tīmekļa rīks MySQL un MariaDB datubāzu pārvaldībai.
- Mērķis: Nodrošināt grafisko interfeisu datubāzes administrēšanai.
- Izmaksas: Bezmaksas.
- Īpašības: Intuitīvs interfeiss, atbalsta plašu MySQL un MariaDB funkcionalitāti, viegla instalācija un lietošana.

### 3. Iespējamais kvalifikācijas darba uzdevums

Aktualitātes pamatojums:
Projekta galvenais mērķis ir radīt lietotājam draudzīgu tīmekļa vietni, kas atvieglos automašīnu meklēšanu un pārdošanu tiešsaistē. Ņemot vērā nepārtraukto automašīnu pieprasījumu un nepieciešamību rīkoties ar plašu informāciju par pieejamajiem auto piedāvājumiem, šāda vietne būs ļoti aktuāla. Tā tiks nodrošināta personām vecumā no 18 gadiem ar transportlīdzekļa vadītāja apliecību, arī tiem, kuri nav ļoti labi orientējušies automobiļu markās.

Uzdevuma nostādne:

- Izveidot tīmekļa lietotni, kas nodrošina efektīvu automašīnu meklēšanu un filtrēšanu.
- Nodrošināt lietotājiem vienkāršu un ērtu veidu, kā izvēlēties automašīnu, kas atbilst viņu vajadzībām un budžetam.
- Ieviešot funkcionalitāti, kas atbilst mūsdienu lietotāju prasībām un vēlmēm.

Tika plānotas vairākas funkcijas:
- detalizētas informācijas  par automašīnu apskate un nepieciešamās preces meklēšana pēc atslēgvārdiem un filtrēšanas kritērijiem; 
- apskatīt piedāvājumu visas pieejamās konfigurācijas. Pievienot vai dzēst konfigurāciju variantus. Paslēpt/deaktivēt konfigurāciju.;
- piedavājumu pievienošana vai dzēšana un informācijas rediģēšanas, piemēram, mainīt cenu, krāsu un citus datus par piedāvājumu;
- veikt pasūtījumu. Apskatīt visus savus pasūtījumus un izvadīt excel tabulā. Apkalpojošajam personālām būs iespēja apskatīt visus pasūtījumus un dzēst tos;
- apkalpojošs personāls noteiks pasūtījuma statusu un pasūtījuma izpildes procesā mainīs pasūtījuma izpildes statusu;
- atbildēt uz jautājumu vai uzdot savu atsevišķajā lapā un dzēst tos. Apkalpojošajam personālām būs iespēja dzēst jebkuru komentāru;
- iespēja mainīt sava profila datus, tādus kā: profila attēlu un paroli. Apkalpojošajam personālām būs iespēja mainīt tikai attēlu;
- iespēja izvēlēties par kādām tēmām saņemt ziņojumus;
- reģistrācija un autorizācija.


Realizācijai nepieciešamie tehniskie un programmatūras līdzekļi:

HTML, CSS, JavaScript: Frontend izstrādei un lietotāja saskarnes veidošanai.
Bootstrap: Responsīva dizaina nodrošināšanai.
SweetAlert: Modernu un pielāgojamu paziņojumu logu veidošanai.
PHP: Servera puses loģikas izstrādei.
Laragon: Lokālās izstrādes vides nodrošināšanai, kas ietver Apache, MySQL, PHP un phpMyAdmin.
Visual Studio Code: Koda rediģēšanai un pārvaldībai.
GIT: Versiju kontroles sistēma, lai pārvaldītu koda izmaiņas.
Notepad++: Teksta un koda redaktors.

### 4. Secinājumi

Paveiktais darbs
Prakses laikā veiksmīgi izstrādāju vairākus projektus, izmantojot Laravel framework'u, un ieguvu praktisku pieredzi datubāzes pārvaldībā ar SQL. Projekti ietvēra tīmekļa lietotņu izstrādi, kurās bija nepieciešams veidot un pārvaldīt datubāzes struktūras, rakstīt un optimizēt SQL vaicājumus, kā arī integrēt datubāzes ar Laravel aplikācijām. Turklāt es apguvu arī attālinātu serveru pārvaldību ar winSCP, kas ietvēra failu augšupielādi, lejupielādi un servera konfigurācijas uzlabošanu. Visos projektos tika izmantoti arī tādi rīki kā Visual Studio Code, Notepad++, Composer, un vairākas Laravel paplašinājumi, piemēram, PHP Namespace Resolver un Laravel Blade Snippets, lai uzlabotu koda kvalitāti un izstrādes efektivitāti.

Iegūtā pieredze
Esmu ieguvis vērtīgu pieredzi darbā ar dažādām programmēšanas tehnoloģijām un rīkiem. Praktiskās iemaņas, kuras esmu ieguvis, ietver:

- Laravel framework izmantošanu efektīvai tīmekļa lietotņu izstrādei.
- Datubāzu pārvaldību, izmantojot SQL, iekļaujot vaicājumu rakstīšanu un optimizēšanu.
- Attālinātu serveru pārvaldību ar winSCP, nodrošinot drošu failu pārraidi un servera konfigurācijas.
- HTML, CSS, JavaScript un Bootstrap izmantošanu, lai izveidotu estētiski pievilcīgas un funkcionālas tīmekļa vietnes.
- GIT versiju kontroles sistēmas izmantošanu koda pārvaldībai un kolaborācijai.
- Visual Studio Code un Notepad++ izmantošanu kā efektīvus koda redaktorus, kā arī Composer rīka izmantošanu PHP projektu atkarību pārvaldībai.
- Papildus prakses laikā apguvu arī Laravel paplašinājumu, piemēram, PHP Namespace Resolver un Laravel Blade Snippets, izmantošanu, lai uzlabotu koda struktūru un produktivitāti.

Zināšanu pielietojums
Tehnikumā iegūtās zināšanas un prasmes bija pietiekamas, lai veiksmīgi izpildītu prakses uzdevumus. Tomēr prakse sniedza iespēju paplašināt šīs zināšanas un iegūt jaunas prasmes, kas būs noderīgas turpmākajā darbā. Praktiskā pieredze darbā ar Laravel framework'u, SQL un citiem rīkiem ļāva man padziļināt izpratni par tīmekļa lietotņu izstrādi un datubāzu pārvaldību. Šīs prasmes ir būtiskas manas profesionālās karjeras attīstībai un nodrošinās, ka es spēšu efektīvi risināt sarežģītus uzdevumus nākotnē. Prakse arī palīdzēja attīstīt spēju strādāt komandā, efektīvi komunicēt un risināt problēmas, kas ir svarīgas prasmes jebkurā profesionālajā vidē.
